# 価値中心ルーブリック (E8-4 / FR-SL-03)

PdM 成果物を LLM-as-Judge で採点する際の観点。各観点 1〜5 点。

## 観点

1. **value_for_whom（誰にとっての価値か）**
   - 5: 具体的なユーザーセグメントと得られる価値が明示されている
   - 3: 価値の方向性は示されているが対象が曖昧
   - 1: 価値の受益者が不明確

2. **output_outcome_value_distinction（アウトプット/アウトカム/価値の区別）**
   - 5: 成果物・アウトカム・ビジネス価値が明確に区別されている
   - 3: 一部のみ区別されている
   - 1: 混同している

3. **tradeoff_clarity（トレードオフの明示）**
   - 5: 主要なトレードオフと判断理由が明示されている
   - 3: トレードオフに触れているが浅い
   - 1: トレードオフが欠落している

## 出力形式

JSON のみ:

```json
{
  "value_for_whom": 4,
  "output_outcome_value_distinction": 3,
  "tradeoff_clarity": 5
}
```
